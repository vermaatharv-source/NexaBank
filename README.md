# NexaBank 🏦

NexaBank is a comprehensive Banking Management System designed to provide a modern, user-friendly banking experience. This project serves as a bridge between foundational backend logic and responsive web interface design, transforming a traditional C++ console application into an interactive web-based platform.

## 🚀 Features

### For Members
- **Secure Authentication:** User-specific login to access account details.
- **Transaction Hub:** Perform seamless cash deposits, withdrawals, and fund transfers.
- **Loan Services:** Apply for loans directly through the portal and track application status.
- **Account Overview:** Real-time balance display and comprehensive transaction history.

### For Admins
- **Full Control:** Manage member databases (add/delete accounts).
- **Loan Approval:** Review and approve or reject pending loan applications.
- **Reporting:** Generate instant branch reports on members, loans, and transactions.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3 (Custom responsive layouts).
- **Logic:** Vanilla JavaScript (Handling state, data processing, and event listeners).
- **Session Management:** Web Storage API (`sessionStorage`) for secure session handling.

## 📂 Project Structure
- `index.html`: Main landing page with login portal.
- `member.html`: Dashboard for members.
- `admin.html`: Management panel for admins.
- `bank.js`: Core logic file where banking functions are processed.
- `*.css`: Stylesheets for UI/UX design.

## 💡 How to Run
1. Clone this repository to your local machine:
   `git clone https://github.com/YOUR_USERNAME/NexaBank.git`
2. Open `index.html` in any modern web browser.
3. Use the provided credentials or create a new account via the admin panel!
